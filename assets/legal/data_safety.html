ChronoSpark Data Safety

Last updated: 2026-06-28

This page summarizes how ChronoSpark handles user data for Google Play Data Safety disclosure alignment.

Data Collected

  • Account identifiers (email) for authentication.
  • User-created app content (tasks, notes, reflections, settings).
  • Purchase/subscription status for premium feature access.
  • Basic diagnostics for app reliability and troubleshooting.

Data Sharing

We do not sell personal data. Data is shared only with service providers necessary to operate the app (for example, authentication, billing, and hosting services).

Security Practices

  • Data in transit is protected using HTTPS.
  • Access to account data is limited to authorized systems and workflows.
  • Users can request account deletion at Delete Account.

Contact

For data safety or privacy questions, contact: privacy@chronospark.app
