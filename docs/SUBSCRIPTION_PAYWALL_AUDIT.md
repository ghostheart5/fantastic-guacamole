# Subscription and Paywall Audit

Because you have in-app purchase packages:

1. ✅ Products created in Google Play Console
2. ✅ Product IDs match app code
3. ✅ Purchase flow works
4. ✅ Restore purchases works
5. ✅ Entitlement check exists
6. ✅ Premium features locked/unlocked correctly
7. ✅ Failed purchase handled

If premium is not ready yet, make sure paywall buttons do not break the app.
