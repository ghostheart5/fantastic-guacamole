# Timeline and Milestone Audit

## Timeline or Milestone Entities Needed

ChronoSpark needs:

1. TimelineEvent
2. Milestone
3. GoalDeadline
4. ProjectDeadline
5. HabitHistory
6. StreakMilestone
7. FutureSelfTimeline
8. TaskDeadline

## Timeline Must Answer

1. What happened?
2. What is happening now?
3. What is next?
4. What is overdue?
5. What is at risk?
6. What is coming soon?
7. What needs attention today?

## Milestones Must Support

1. Create Milestone
2. Complete Milestone
3. Track Progress
4. Forecast Completion
5. Detect Risk
6. Celebrate Achievement
7. Replan Milestone

## Audit Checks

1. ✅ Can timeline query real event history?
2. ✅ Can timeline identify current, next, and overdue items?
3. ✅ Can timeline compute risk and near-term deadlines?
4. ✅ Can milestones track creation, progress, completion, and archive lifecycle?
5. ✅ Can milestones forecast completion and flag at-risk milestones?
6. ✅ Can milestone completion trigger celebration and timeline mirroring?
